# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest on `main` | Yes |
| Older releases | No |

## Reporting a Vulnerability

**Please do NOT open a public Issue for security vulnerabilities.**

Please report security issues through [GitHub's private vulnerability reporting](https://github.com/Tiryoh/rosbag-analyzer-web/security).

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond as soon as possible and work with you to address the issue.

---

# セキュリティポリシー

## サポート対象バージョン

| バージョン | サポート |
|-----------|---------|
| `main` の最新版 | はい |
| 過去のリリース | いいえ |

## 脆弱性の報告

**セキュリティ上の脆弱性については、公開 Issue を作成しないでください。**

[GitHub のプライベート脆弱性報告機能](https://github.com/Tiryoh/rosbag-analyzer-web/security) から報告してください。

以下の情報を含めてください:
- 脆弱性の説明
- 再現手順
- 想定される影響

できる限り速やかに対応いたします。
